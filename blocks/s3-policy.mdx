title: 'S3 Access from Anywhere Policy'
description: 'This policy allows the users with the bucket credentials to perform all CRUD actions inside of your bucket.'
icon: 'code'
---

### S3 Acess from Anywhere Policy

```json 
{
    "Version": "2012-10-17",
    "Statement": [
        {
            "Sid": "PublicReadGetObject",
            "Effect": "Allow",
            "Principal": "*",
            "Action": "s3:GetObject",
            "Resource": "arn:aws:s3:::landdepot-root-bucket/*"
        }
    ]
}
```
